Here is a list of open-source projects and works in progress. All source code is hosted on [GitHub](https://github.com/xaashay).

## FPGA Projects

### Clock Divider Module
A parameterizable clock divider in Verilog supporting both even and odd division ratios. Includes a full simulation testbench.

### D Flip-Flop Collection
A collection of basic sequential building blocks — D flip-flops, T flip-flops, JK flip-flops — with simulation testbenches.
